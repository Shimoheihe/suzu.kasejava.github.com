Styling your headers

You can style it in your css using the class .kaseiiki.your-class.

Use whatever CSS or markup you like for each one. They will be switched automatically when necessary.

.kaseiiki.default {
  background: none;
  color: black;
}
.kaseiiki.white {
  background: white;
  color: black;
}
.kaseiiki.blue {
  background: blue;
  color: white;
}

The jQuery Plugin

To make everything work, just load energy (and jQuery) and initialize it. We take care of the rest.

Further customization

Want to use custom markup for each header?

You just need to add a div with the class .kaseiiki.default inside your header, as well as any other custom headers you want.

For instance the "butterflies" section will use the .kaseiiki.butterflies header, and any other ones will just use the markup from .kaseiiki.default.
